# 用語集

## ビジネス用語

| 用語 | 英語 | 説明 |
|------|------|------|
| 投稿 | Post | ユーザーが作成する旅行記録。タイトル、本文、画像、位置情報、タグを含む |
| ブックマーク | Bookmark | ユーザーが後で見返すために保存した投稿 |
| フォロー | Follow | 他のユーザーの投稿を追跡する機能 |
| フォロワー | Follower | 自分をフォローしているユーザー |
| フォロイー | Following | 自分がフォローしているユーザー |
| タグ | Tag | 投稿を分類するためのラベル |
| 位置情報 | Location | 投稿に紐づく場所の情報（緯度、経度、地名） |
| 公開投稿 | Public Post | 全ユーザーが閲覧可能な投稿 |
| 非公開投稿 | Private Post | 投稿者本人のみが閲覧可能な投稿 |

## 技術用語

| 用語 | 説明 |
|------|------|
| JWT | JSON Web Token。認証トークンの形式 |
| OAuth | 外部サービスを利用した認証プロトコル |
| API | Application Programming Interface。システム間通信の仕様 |
| REST | Representational State Transfer。API設計の原則 |
| CRUD | Create, Read, Update, Delete。基本的なデータ操作 |
| ORM | Object-Relational Mapping。オブジェクトとDBの変換 |
| SSR | Server Side Rendering。サーバー側でHTMLを生成 |
| CSR | Client Side Rendering。クライアント側でHTMLを生成 |
